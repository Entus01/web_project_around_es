# Tripleten web_project_around_es

En este proyecto se crea una pagina web para que un usuario pueda personalizar su imagen de perfil, su nombre, una descripcion de si mismo. De igual manera el usuario tiene la opcion de agregar imagenes a su perfil, podrá darles "like", podra eliminarlas e incluso mostrar las imagenes en tamaño original al hacer click sobre ellas. Todos los cambios realizados seran enviados por medio de APIs a un servidor que guardara los cambios. En cada uno de los formularios la pagina mostrará cuando los cambios estén guardandose, una vez esto cambios se ejecuten, la pagina cerrará la ventana emergente donde se realizaron los cambios. De abrirse cualquiera de las ventanas emergentes nuevaqmentes, estas estaran vacias, a excepción de la ventana de cambio de nombre de perfil y descripción, que los espacios mostrarán los valores actuales del usuario.

Herramientas utilizadas:
POO
Clases y subclases
Importar/ Exportar entre Modulos
Manejo de formularios
Programacion asincrona
Metodos HTTP
APIs
Control de versiones
Estilos con CSS
